### Base32编码
`codec.base32Encode(obj)`

#### 参数

| 名称  | 说明    | 类型     | 必填  |
|-----|-------|--------|-----|
| obj | 目标对象  | Object | 是   |

#### 返回值
- 类型：**String**
- Base32字符串

### Base32解码
`codec.base32Decode(str)`

#### 参数

| 名称  | 说明        | 类型     | 必填  |
|-----|-----------|--------|-----|
| str | Base32字符串 | String | 是   |

#### 返回值
- 类型：**String**
- 源字符串
